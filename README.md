# ROMR Engineering Knowledge Platform

MkDocs Material based integrated engineering knowledge platform for the AI-Based Autonomous Road Marking Platform.

## No-terminal deployment path

1. Create a GitHub repository.
2. Upload all files and folders in this package to the repository root.
3. Go to **Settings → Pages**.
4. Select **Build and deployment → GitHub Actions**.
5. The included workflow will publish the site automatically.

Main configuration file: `mkdocs.yml`.
